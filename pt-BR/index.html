Workspace

Tudo aparece como uma prateleira de apps, mas é construído como um lab.

A Libremotor separa as escolhas do usuário dos detalhes técnicos de cada build. Escolha a capacidade, revise os termos e deixe as ferramentas montarem o perfil certo.

Hub

Um lugar para ativação, escolha de perfil, recuperação e status.

Incluído

Como funciona

Escolhas claras, saída repetível.

  1. 01

    Solicite acesso

    Comece com um teste por tempo limitado ou envie os detalhes necessários para aprovação no beta privado.

  2. 02

    Ative o Hub

    O Hub checa elegibilidade e mantém as escolhas vinculadas ao dono.

  3. 03

    Escolha um perfil

    Selecione navegação, vídeo, projeção ou um setup menor de recuperação.

  4. 04

    Compile e verifique

    Receitas produzem APKs assinados com checagens contra saídas incorretas.

Beta privado

Feito para donos cuidadosos, não para instalações casuais.

A Libremotor é software experimental para hardware do próprio usuário. O beta atual foca em instalações guiadas, termos explícitos e fluxos reversíveis antes de uma abertura maior.
